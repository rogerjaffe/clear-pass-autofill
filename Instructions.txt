ClearPass autofill the "No"s

1. Open Google Chrome
2. Click the settings menu (three vertical dots in the top right corner)
3. Click More tools | Extensions
4. Check Developer mode
5. Click Load unpacked extension...
6. Choose the ExtensionFiles folders located in the same folder as 
   these instructions.
7. The extension will load into Chrome
8. Uncheck Developer mode box
9. In the ClearPassAutofill extension box check Allow in Incognito
10. You can delete these files when you're finished

Now when you go to any attendance page, the column width for the student name will
be wide enough to display almost all student names on one line!

You can disable or delete this extension any time from the Chrome extensions page
